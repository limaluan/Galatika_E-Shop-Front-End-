## 🚧 Projeto
 Esse projeto trata-se de um e-commerce com funcionalidades como:
 - Sistema Completo de Registro e Autenticação do Usuário.
 - Criação e Gerenciamento de Produtos.
 - Carrinho do Usuário.
 - Criação de Seções.
